üéØ Built for Audio Engineers

Alioop solves the real-world problem of delivering finished audio files to clients. No more emailing huge files, managing Google Drive links, or chasing payments. Just drop the file, select the client, and you're done.


Two delivery methods: Web portal for quick uploads with beautiful UI, or File watcher for automated processing when files appear in a folder.

# Clone the repository
